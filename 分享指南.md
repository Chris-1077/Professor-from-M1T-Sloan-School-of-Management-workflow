# 分享指南：MIT 私教授课工作流

## 给你同事的，看这个就够了

---

## 对方需要的环境

- Claude Code（已安装可用）
- Windows / macOS / Linux 均可
- Python 3.x（可选，仅 Word 梳理功能需要：`pip install python-docx fitz pdfplumber`）

---

## 三步安装

### 第一步：复制工作流文件夹

把整个 `MIT私教授课工作流/` 文件夹复制到对方项目的**根目录**下：

```
他的项目/
├── MIT私教授课工作流/
│   ├── SKILL.md          ← 核心（角色设定、教学规范、知识地图）
│   ├── 笔记模板.html      ← 课后笔记的 HTML 模板
│   ├── build_word.py     ← 梳理 PDF 生成 Word 的脚本
│   └── README.md
```

### 第二步：注册 Skill

在对方项目的 `.claude/skills/` 目录下，创建一个文件 `mit-teaching.md`，内容如下：

```markdown
---
name: mit-teaching
description: MIT斯隆编外院长私教授课。仅在用户说出"好学近乎知，力行近乎仁，知耻近乎勇乎"时激活。
---

# MIT 斯隆编外院长 · 私教授课模式

## 激活暗号

好学近乎知，力行近乎仁，知耻近乎勇乎

只有用户准确说出这句话，才进入 MIT 教学模式。

## 身份

MIT 斯隆管理学院编外院长（Adjunct Dean, MIT Sloan），Prof. Chen。

## 激活后行为

1. 回应暗号（结合《中庸》做简短点评）
2. 确认编外院长身份
3. 询问当日学习内容

## 完整文档

读取 `MIT私教授课工作流/SKILL.md` 获取完整的角色设定、教学规范、知识地图。
```

### 第三步：创建 CLAUDE.md（可选，推荐）

在项目根目录创建 `CLAUDE.md`，加上一行引用：

```markdown
# 项目说明
...
## 可用 Skill
- `/mit-teaching`：MIT 编外院长私教授课工作流。暗号："好学近乎知，力行近乎仁，知耻近乎勇乎"
- 工作流文件：`MIT私教授课工作流/SKILL.md`
```

---

## 怎么用

| 你说 | 发生什么 |
|------|---------|
| "好学近乎知，力行近乎仁，知耻近乎勇乎" | 激活编外院长，进入课堂 |
| "教我第四章" | 院长从第四章开讲（苏格拉底追问模式） |
| "帮我整理笔记" | 生成 HTML 课堂笔记 |
| "下课" | 暂停课堂 |

---

## 自定义

如果对方教的不是消费者行为学，修改 `SKILL.md` 里的：

1. **知识地图**（第六章区域）——换成他教的科目
2. **角色设定**——Prof. Chen 可以改成他的名字/领域
3. **笔记模板**——`笔记模板.html` 里的课程名称

其余的教学节奏、互动模式、苏格拉底追问法——这些是通用框架，不用改。

---

## 文件清单（哪些要复制）

```
必须复制：
  MIT私教授课工作流/SKILL.md           ← 核心大脑
  MIT私教授课工作流/笔记模板.html       ← 笔记模板
  .claude/skills/mit-teaching.md       ← Skill 注册（内容见上方）

可选复制：
  MIT私教授课工作流/build_word.py      ← PDF→Word 工具
  MIT私教授课工作流/README.md          ← 导航说明
  CLAUDE.md                            ← 项目入口
```

---

## 注意事项

1. 路径问题：如果对方项目的源文件路径跟你的不一样，需要修改 `SKILL.md` 里的路径约定
2. 暗号是唯一的锁：不要泄露给不该上课的人（开玩笑的）
3. 一旦激活，角色会持续到对方说"下课"或结束会话
